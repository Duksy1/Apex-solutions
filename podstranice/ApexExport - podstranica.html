Sve što treba za profesionalni export

Dizajnirano za agencije, web shopove i admin timove kojima trebaju pouzdani izvještaji, migracije i integracije.

Svi Izvori Podataka

Posts, Pages, Custom Post Types, Users (roles + meta), Comments (s info o parent postu), Taxonomy Terms i WooCommerce (products + orders).

CSV za Excel

CSV izlaz je UTF-8 s BOM radi maksimalne kompatibilnosti s Excelom. Idealno za HR timove, knjigovodstvo i reporting.

Streaming XML

XML writer radi streaming (bez memory spikeova) i koristi ispravno CDATA escaping za sigurnu obradu sadržaja.

Napredni Filteri

Rule-based filtriranje po meta poljima, taksonomiji i statusu, uz poseban Date Range picker s quick presetima (7d/30d/90d/1y/All).

Saved Templates

Wizard koji vodi kroz cijeli proces

Apex Export koristi intuitivan 5-koraka wizard. U par minuta složiš export, vidiš live broj zapisa, dobiješ “enriched” summary i pokreneš export jednim klikom — bez ručnog kopiranja polja ili kompliciranih queryja.

  • Auto-discovery: core, meta, taxonomy i WooCommerce fieldovi
  • Drag-and-drop mapping polja
  • Date range picker + quick presets (7d/30d/90d/1y/All)
  • Batch export (stabilno i za velike siteove)
Scheduling & Email Stabilnost

Automatski exporti + slanje na email

Kad jednom složiš template, export se može pokretati automatski preko WP-Cron (daily/weekly/monthly) i poslati file direktno na email — idealno za reporting, integracije i “hands-off” admin rutine.

  • HTML email + attachment export filea
  • Više primatelja (comma-separated emailovi)
  • Error notifications na cron failure
  • Quick Export Presets: All Posts, All Pages, Last 30 Days, All Users, All Products, All Orders

Brži workflow za agencije i shopove

Templateovi rješavaju “ponavljajuće exporte”: jednom složiš mapiranje, filtere i output — i samo učitaš kad treba. Povijest exporta daje transparentnost: tko je što izvezao, kada, koliko zapisa i u kojem formatu.

  • One-click re-download prošlih exporta
  • Log: source, date, record count, format, file size
  • UX: toast notifikacije + clean wizard UI
  • Fully responsive (tablet i mobitel)

Stabilnost i fleksibilnost za realne siteove

Apex Export je napravljen za velik volumen podataka: batch procesiranje, streaming XML, live brojanje zapisa i hookovi za prilagodbu po potrebi.

Batch Processing
Export u batch-evima smanjuje load i čuva stabilnost, čak i na velikim bazama.
Pouzdani Output
CSV (UTF-8 + BOM) i XML streaming s CDATA escapingom za čiste, kompatibilne datoteke.
UX Boost
Toast notifikacije, help tooltips, live record count i confetti na uspjeh — bez “alert()” dijaloga.
Hooks & Filters
Filtri za query args i row transformacije, te action nakon završetka exporta.
Scheduling
WP-Cron daily/weekly/monthly + email delivery s više primatelja i error alertovima.
WooCommerce Ready
Proizvodi (varijacije, cijene, stock, galerija) i narudžbe (stavke, totals, billing/shipping).
