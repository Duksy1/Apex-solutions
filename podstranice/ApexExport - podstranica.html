Everything You Need for Professional Exports

Designed for agencies, web shops, and admin teams who need reliable reports, migrations, and integrations.

All Data Sources

Posts, Pages, Custom Post Types, Users (roles + meta), Comments (with parent post info), Taxonomy Terms, and WooCommerce (products + orders).

CSV for Excel

CSV output is UTF-8 with BOM for maximum Excel compatibility. Ideal for HR teams, accounting, and reporting.

Streaming XML

The XML writer uses streaming (no memory spikes) with proper CDATA escaping for safe content processing.

Advanced Filters

Rule-based filtering by meta fields, taxonomy, and status, plus a dedicated Date Range picker with quick presets (7d/30d/90d/1y/All).

Saved Templates

Save any export configuration as a template and load it in a few clicks. Templates are stored as CPT (apex_export_tpl).

Export History

Log of every export (source, date, record count, format, file size). One-click re-download of past exports. Table: apex_export_history.

A Wizard That Guides You Through the Entire Process

Apex Export uses an intuitive 5-step wizard. In a few minutes you set up an export, see the live record count, get an "enriched" summary, and launch the export with one click — no manual field copying or complicated queries needed.

  • Auto-discovery: core, meta, taxonomy, and WooCommerce fields
  • Drag-and-drop field mapping
  • Date range picker + quick presets (7d/30d/90d/1y/All)
  • Batch export (stable even for large sites)
Scheduling & Email Stability

Automated Exports + Email Delivery

Once you set up a template, exports can run automatically via WP-Cron (daily/weekly/monthly) and send the file directly by email — ideal for reporting, integrations, and "hands-off" admin routines.

  • HTML email + file attachment
  • Multiple recipients (comma-separated emails)
  • Error notifications on cron failure
  • Quick Export Presets: All Posts, All Pages, Last 30 Days, All Users, All Products, All Orders
