Immersive Ad Formats

Parallax Interscroller Lazy Scripts

Beyond Standard Banners

Parallax ads use configurable data-speed (0.05–1) for subtle or dramatic scroll effects. Interscroller ads stick between content sections and reveal CTA buttons when triggered by IntersectionObserver.

  • Parallax: configurable height, speed, and background image
  • Interscroller: sticky creative with viewport-triggered CTA
  • Scripts loaded conditionally — only on pages with these ads
  • Lower speed values = more subtle, cinematic effect
  • IntersectionObserver for battery-efficient viewport detection

REST API for Headless Sites

Fetch rendered ad HTML via REST endpoints for headless WordPress, SPAs, or AJAX-based layouts. Input is validated, sanitized, and group slugs are regex-restricted.

  • /wp-json/sam/v1/ads/<id> — single ad HTML
  • /wp-json/sam/v1/groups/<slug> — group ads
  • ID params: validate_callback + sanitize_callback
  • Group slug regex: [a-zA-Z0-9\-_,]+, max 10 per request
  • Returns pre-rendered HTML ready for AJAX injection
// Fetch single ad via REST
fetch( '/wp-json/sam/v1/ads/42' )
  .then( r => r.json() )
  .then( data => {
    container.innerHTML = data.html;
  });

// PHP helper
echo sam()->service('renderer')
  ->render( 42 );

Security and Performance Built In

Every click is verified, every export is protected, and every query is optimized. No shortcuts on security.

Nonce-Protected Clicks
Every tracking URL contains a WordPress nonce verified before registering a click. Prevents forged click requests and bot inflation.
Rate Limiting
Transient-based IP throttling — max 1 click per ad per IP per minute. Blocks click fraud without external dependencies.
XSS & CSRF Protection
CSS classes through sanitize_html_class(), AdSense slots regex-sanitized, CSV export with check_admin_referer().
Iframe Sandboxing
Video embeds use sandbox="allow-scripts allow-same-origin allow-popups" with referrerpolicy="no-referrer".
Optimized Queries
Dedicated _sam_auto_insert meta key for auto-insert lookups — no LIKE on serialized data. Max 50 ads per query.
REMOTE_ADDR Only
Client IP read exclusively from REMOTE_ADDR — no insecure X-Forwarded-For headers. Reliable rate limiting you can trust.
